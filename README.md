Github Projects を使い易くするための拡張
========================

概要
-----

## できること
- タスクボード上でIssueを選択した時、子Issueが有れば右ペインの詳細に表示する
- タスクボード上でIssueを選択した時、子Issueがボード上有れば子Issueをハイライトする

## インストール方法
- chrome://extensions/ にアクセス
- `デベロッパーモード` をONにする
- `パッケージ化されていない拡張機能を読み込む` でsrcフォルダを指定
